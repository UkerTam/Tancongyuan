产品概述

DESCRIPTION

产品定位 /

艾三维协同设计管理app——CDMS APP是面向企业、建筑设计单位存储管理资料文件的平台。

项目背景 /

基于艾三维协同设计平台web端功能所开发的手机应用，用户所操作的对象主要是文件、流程、操作模型。CDMS以项目文档共享为核心，以文档过程控制为主线，实现项目文档全过程管理。完善的文档权限控制技术和加密技术，最大限度保障企业数据的安全。同时还拥有模型操作功能，使建筑设计单位实现对模型随时随地在线预览。

用户画像

USER PORTRAIT

用户画像由带有特征的标签组成，通过标签我们可以更好地理解谁是这个产品的主要用户。

建筑设计院属于知识密集型企业，生成成品是电子文档，所以如果有效的管理控制设计成品，是设计院信息化要解决的核心问题。 经过统计，设计人员30%的时间是用于设计相关资料的查找和收集，所以对于设计院来说，实现各种文档的共享和设计人员工作之间的协同设计，是提高设计效率的唯一手段。

王碧莲

路桥工程师

业主或协作单位提交的数据都是直接提交给项目组的某个设计人员，而其它相关设计人员则需要间接获取数据，这种接力式的数据传递，很难保证数据的准确传达。


曾文涛

设计院管理

设计过程中产生大量重复性或相似性的电子图档，极易造成管理上的混乱，需要定制流程。在出差驻场时想要了解模型建造情况，希望只需通过手机即能浏览模型。

标签栏图标

文档

收录企业所有文件的界面，用户能查看所有目录，可以在自己权限范围内对文件进行操作。

设计风格为大留白设计，去除多余分割线，通过留白区分层级。这样的好处是可以减少分割线对内容的干扰，界面呼吸感增强，达到简约设计的目的。

*点击手机轮播后将切换为手动滑动swiper

新建

新建作为此软件的核心功能之一，利用悬浮按钮的设计可以引导不熟悉界面的用户来探索。悬浮按钮通常用来承载最常用、最重要的操作，并且主要以正向操作如创建为主。

弹出层采用白色覆盖全屏，利用高饱和渐变色作为底色，配上大圆角面性图标，使其更容易区分功能，更利于用户进行决策。

更多操作

点击目录/文件右侧的图标，由底部弹出更多操作的浮层，可下拉或点击黑色区域将其关闭。

权限设置

流程

文档的处理，往往需要经过一定的流程; 处理过程中可能会经过不同的人员，不同的阶段的每个人员对文档所进行的处理可能有所不同; 例如建筑图纸的设计、校对、审核、批准等的处理。

利用流程模板，用户可为指定的文档或文件选择任一定义好的流程模板启动具体的流程。
